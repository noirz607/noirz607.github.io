# Another Page
>但有时候我也在想：  
要是我也能经历这样的青春的话；  
要是我的面前有一个泪眼迷糊的女主角的话；  
要是我也是轻小说的主角的话；  
那时……
我又会想什么呢？

{% biliplayer BV1D64y187w8 %}

真是莫名奇妙，我为什么要放这个链接，再引用这个中二到不知所云的引用呢？难道我还是幻想自己是世界的主角的高中生吗？！  

这本来是一个测试用的文档，用来练习md的语法以及hexo主题提供的一些功能。可当我准备删掉的时候却突然想： 

***“要不把这个当作我博客的第一篇文章吧！”***  

于是乎，我便坐下来写了这些文字。就像我不会删去我电脑里的那段helloworld一样，我也打算把这些东西保留下来，或许哪天看到这些的时候，总有一些不一样的感触。   

如果问高一的我，你会不会搭一个博客，把你所思所想放到网上。他肯定会回答我：“什么鬼东西，感觉好麻烦，写日记这样的事情也太没有意义了！” ~~搞得像你未来三年能干出什么有意义的事情一样XD~~  

好像的确如此：我从来没有记笔记的习惯，也鲜有拍照记录下自己的生活。或许的确没有什么可记录的。上学，看动漫，打游戏。我只会在这三个状态来回切换。得过且过 或许可以如此概括我的18年人生。  
  
我问我同学，为什么我没有天降女主角关心我，为什么别人的高中生活好像如此丰富多彩？

{% asset_img image1.jpg %}  

大概就是这么回应我的 ~~（没这么好看）~~  

可他的确什么都没说错，我并不是轻小说的主角，更不会有一个泪眼迷糊的女主角为我哭泣。我只是一个普通到扔在人海里都找不到的宅男，和大多数人一样，幻想着自己成为世界的中心，手上却不做出任何改变。可当我切切实实地踏入大学的校门，我才意识到那些荧幕里我曾向往过的青春校园，已然遥不可及。

前些日子是校运动会，而举办的校区就在我高中边上。“回去看看吧！”自然而然的这么想。

被残忍的拒之门外了。也是，保安不会让一个不穿校服还拿着手机的人进学校。

可这又怎么难得到我！我立刻回想起来一条高三的时候偷偷出门吃kfc的神秘路线，装作来看场地的羽球球友大摇大摆地走进游泳馆二楼，转头溜进楼梯间，快跑进教学楼。

这里的确没有任何变化，除了高一高二搬到了另一栋楼，一切都和我离开的时候一样：楼梯间里被扣掉的宣传语，一楼吵吵闹闹的小学生，书柜上莫名其妙的一大堆试卷。她就在那里，只是我已经离开了，

多么讽刺，我的高中是我这三年的全部，可我只是这他的一个匆匆过客。

骑车回程的路上，我再次经过了那条贯穿整个城镇的铁路。

{% asset_img image2.jpg %} 

我一直很好奇，为什么这条铁路一直留在这里，明明他不承载任何线路，更不会有任何一辆机车经过这里。

可他就是在这里，或许是出于战时的考量，也只是单纯的拆除费用太高，他一直留了下来。

>莫名其妙，这和我文章的主题有任何关系吗?  
我的文笔实在是差的可以，让chatgpt来都能写点比这好得多的文字。

可我终究不是一块枕木，一条铁轨。

大学生活是奇特的。没有目标，没有标准，没有你想要就有的教学和答案，没有人在乎你。只有冰冷的矩阵和奇异的高数构造在冲击我的价值观。

我不知道未来我会怎么样，一个几个月前还要对着教程打helloworld的人能成为一个优秀的人吗？

对着电脑屏幕发呆。我的高中同学已经不再我身边了，414里神秘的八卦聊天环节早已在两年前就结束了。留下的只有一个孤独且恐惧的空壳，思考这毫无意义的议题，等待着有一个普普通通的明天。

“进入大学的殿堂，让我们翻开崭新的一页。”在熟悉不过的开场词。**“Another Page”** 我该往里面写点什么呢？

或许留着空白，等我有了头绪再写也不迟吧。

没有再伤感的理由了，明天已然到来。  
纵使普普通通，也是要交数论作业的。
